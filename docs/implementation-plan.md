# 実装計画

## 方針

- `github.md` のタスク・コミット履歴をデモデータとして再現する。
- GitHub MCP 連係の本体は差し替え可能なアダプタとして実装する。
- Chat では、issue と commit の両方からメンバーのスキルを推定し、回答に使う。

## 今回のデモで扱う対象

- メンバー: 中村 大樹 / 前田 彩 / 小林 拓海 / 鈴木 莉子 / 山田 花奈 / 田中 誠
- 主要ソース: issue, commit, task note, repository summary
- 出力: スキル要約, 根拠付きプロフィール, チャット用の推薦文

## 開発ステップ

1. データモデルとデモデータを定義する。
2. GitHub MCP 互換のローカルアダプタを実装する。
3. issue / commit からスキル信号を抽出する。
4. 正規化と重み付けを追加する。
5. ハイブリッドスコアを組み合わせる。
6. Chat 応答と CLI / API の入口を作る。
7. 評価用の集計とサンプル出力を整える。
8. README とテストを仕上げる。
